> umi(react+router)+dva+typescrip开发脚手架

## 脚手架特性

1. TypeScript的支持
2. 使用Dva进行状态管理，是redux+redux-saga的最佳实践之一
3. 集中式声明式路由管理，基于react-router实现，集成在umi
4. 异步加载和按需加载，集成在umi
5. less替换css，集成在umi
6. 登陆Layout和面板Layout的布局设置
7. 使用immutable.js优化redux
